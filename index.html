Sobre Mim

Olá, meu nome é Caian Patrick e sou um desenvolvedor Front-End apaixonado por criar experiências digitais incríveis. Tenho experiência em HTML, CSS e JavaScript, e estou sempre em busca de aprender novas tecnologias e aprimorar minhas habilidades. Meu objetivo é desenvolver interfaces de usuário intuitivas e responsivas que proporcionem uma navegação fluida e agradável.

Meus Projetos

Site De Uma Cafeteria

Este site de cafeteria foi desenvolvido utilizando HTML, CSS e JavaScript, oferecendo uma experiência moderna e responsiva para os visitantes. A estrutura HTML organiza o conteúdo de forma clara e acessível, enquanto o CSS estiliza as páginas com um design acolhedor, inspirado em cafeterias artesanais. O JavaScript adiciona interatividade ao site, como um menu dinâmico, sistema de filtro de produtos e um modo noturno opcional. Ideal para apresentar o cardápio, horários de funcionamento, localização e promoções especiais da cafeteria.

Loja Virtual

O Dev Store é um projeto de loja virtual desenvolvido com HTML, CSS e JavaScript, oferecendo uma experiência de navegação moderna, responsiva e centrada no usuário. O site simula um e-commerce com produtos exibidos em cards organizados, visual atraente e navegação fluida. O layout é limpo e bem estruturado, com destaque para a vitrine de produtos e uma interface amigável. Funcionalidades como filtros ou organização visual dos itens proporcionam uma base sólida para expansão futura, como carrinho de compras ou integração com back-end.

Pagina De Login

Este projeto consiste em uma página de login desenvolvida exclusivamente com HTML e CSS. A interface apresenta um layout moderno, limpo e centrado, com campos para e-mail e senha, além de um botão de acesso. O design responsivo garante boa visualização em diferentes tamanhos de tela. A aplicação destaca o uso eficaz de estilização com CSS puro, incluindo sombras, bordas arredondadas e alinhamento centralizado, demonstrando uma sólida compreensão de estruturação e design visual mesmo sem o uso de JavaScript.

Sorteador De Números

Este projeto entrega exatamente o que promete: um sorteador de números funcional e intuitivo, desenvolvido com HTML, CSS e JavaScript. Com um visual limpo e amigável, o usuário pode definir um intervalo numérico e, com um clique, obter um número aleatório instantaneamente. A interação é rápida, a resposta é clara, e a experiência é fluida – tudo o que se espera de uma ferramenta leve, feita para resolver um problema específico de forma elegante. Um ótimo exemplo de como poucos recursos bem aplicados podem gerar um resultado funcional e agradável.
